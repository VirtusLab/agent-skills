# VirtusLab Agent Skills

This repository contains a collection of AI agent skills maintained by VirtusLab, following the [Anthropic Agent Skills](https://github.com/anthropics/skills) specification.

## Skills

- [java-best-practices](./java-best-practices): Provides Java best practices and coding standards curated by VirtusLab engineers.

## Usage

To use these skills with a compatible agent (like Claude or Gemini via CLI):

1. Clone this repository.
2. Configure your agent to load skills from this directory.
